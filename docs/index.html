Backend setup (quick reference)

  1. SSH into your HPC login node
  2. Copy backend/server.py to ~/quota-monitor/
  3. Run pip install fastapi uvicorn --user
  4. Edit the config block at the top of server.py
  5. Start: uvicorn server:app --host 0.0.0.0 --port 8000
  6. Or use the systemd service for persistence
  7. Paste the URL above and click Connect
